## ðŸš€ Getting Started

### Prerequisites

**New to coding?** Start with [Lesson 0: Environment Setup](./lesson-0/) for complete instructions on setting up your MacBook from scratch!

**Already set up?** Make sure you have:
- Node.js installed on your computer
- A text editor (VS Code recommended)
- A terminal/command prompt
- Git installed

### Installation

1. Clone this repository
2. Install dependencies:
```bash
npm install
```

### Running Lessons

To run any lesson file:
```bash
npm run lesson lesson-1/01-print-statements.ts
```

Simply replace the filename with the one you want to run!
